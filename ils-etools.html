ILS eTools Overview

ILS eTools are downloadable PDF Quick Reference tools designed to run on any device — desktop, laptop, tablet, or smartphone — using the free Adobe Acrobat Reader. No internet required once downloaded.

  • Africa-MIS Quick Reference eTools — Sub-Sahara Africa focus
  • Guide: Offline Navigation & Tracking
  • Low-Cost Mobile Tools
  • Avenza Maps App
